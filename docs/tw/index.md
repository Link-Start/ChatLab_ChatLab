# ChatLab

ChatLab 是一款免費、開源、本機優先的聊天記錄分析工具。

在數位時代，聊天記錄早已不再只是簡單的文字檔案。它承載著長達十年的社交關係脈絡、親人珍貴的語音片段，更是我們保存在數位世界裡的情感記憶。

ChatLab 的誕生，就是為了**讓每位使用者都能安全地分析、回顧屬於自己的社交記憶。**

## 你可以用 ChatLab 做什麼

- **視覺化分析**：詞頻、詞雲、發言排行、互動熱力圖、活躍時段分布……用圖表看懂你的聊天歷史。
- **AI 對話**：接入 AI 模型後，可以用自然語言提問，讓 AI Agent 協助搜尋、摘要與分析聊天內容。
- **隱私優先**：聊天資料預設在本機儲存與分析；啟用 AI 後，只會將目前問題需要且經過處理的內容傳送給你設定的模型服務商。
- **未來**：ChatLab 的目標是成為社交記憶層的標準基礎設施——沉澱每段關係的記憶檔案，成為 AI Agent 理解你社交世界的本機介面。詳見[專案路線圖](https://chatlab.fun/tw/roadmap/intro)。

## 支援的聊天記錄平台

ChatLab 目前原生支援匯入 QQ、WhatsApp、LINE、Discord、Instagram、Telegram 和 iMessage 的聊天記錄。按照[匯出聊天記錄指南](/tw/usage/how-to-export)準備檔案後，即可直接匯入使用。

如果你使用的平台尚未支援，也可以先匯出聊天記錄，再使用官方 [AI 輔助轉換 Skill](/tw/standard/ai-converter) 將其轉換為 ChatLab Format，轉換完成後即可正常匯入。
